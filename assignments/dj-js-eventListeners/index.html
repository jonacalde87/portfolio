This website was created that a user can change the color of a square.
The square will react to the mouse action the user do.
    The square's color will change as follows:
  • Blue - when the mouse hovers over the square; or user press the letter "b"
  • Red - when the mouse button is held down over the square; or user press the letter "r"
  • Yellow - when the mouse button is let go over the square; or user press the letter "y"
  • Green - when the mouse is double clicked in the square; or user press the letter "g"
  • Orange - when the mouse scroll is used somewhere in the window (not just over the square); or user press the letter "o"
